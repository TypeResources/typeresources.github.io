Tips/help

Settings are stored in your browser so they are there when you reopen it and they are synced with the URL so it's easy to share with friends. Glyph Gazer works with fonts installed on your system and Google Fonts. It works well with the Glyph Gazer Gylphsapp script that opens the content of your current tab in Glyphsapp directly.

Fonts

Write a comma-separated lists of fonts. For Google Fonts you can add e.g. ":900" after the font name to get it in black weight.

OpenType Features

Write a comma-separated list the features you'd like to enable. E.g. salt, ss01, calt, liga, dlig, clig, onum, tnum, smcp, unic, swsh, cswh, zero, sups, subs
